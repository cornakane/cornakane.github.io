Install

  • Go to the Kodi file manager.
  • Click on "Add source"
  • The path for the source is https://cornakane.github.io/ (Give it the name "cornakane Repo").
  • Go to "Addons"
  • In Addons, install an addon from zip. When it asks for the location, select "cornakane Repository", and install repository.cornakane.zip.
  • Restart Kodi
  • Go back to Addons install, but this time, select "Install from repository"
  • Select the "cornakane Repository"
  • Go into the "Video add-ons" section in the repo, and you'll find all video add-ons
